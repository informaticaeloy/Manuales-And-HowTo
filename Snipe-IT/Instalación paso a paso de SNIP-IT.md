### Requisitos iniciales
- VM Ubuntu Desktop 22.04.2 6a bits
- 4GB, 2 CPUs, 30GB Hard Disk

### Installing

```shell
wget https://raw.githubusercontent.com/snipe/snipe-it/master/install.sh
```

```shell
ls -la
```

```shell
chmod 744 install.sh
```

```shell
ls -la
```

```shell
./install.sh
```
